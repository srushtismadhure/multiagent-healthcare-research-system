# Source Quality — Memory Store

**Purpose:** Track which sources are reliable, which are noisy, and any source-specific quirks discovered during research runs.

---

## Tier 1 — High Confidence (Official / Primary Sources)

| Source | URL | Notes |
|--------|-----|-------|
| CMS Newsroom | https://www.cms.gov/newsroom | Official rule announcements; authoritative |
| White House | https://www.whitehouse.gov/presidential-actions | EO text is authoritative |
| FDA | https://www.fda.gov | Device approvals are authoritative |
| BLS | https://www.bls.gov | Labor data — note publication vintage year |
| Federal Register | https://www.federalregister.gov | Proposed and final rule text |

---

## Sources to Avoid or Use with Extreme Caution

| Source | Reason |
|--------|--------|
| Vendor press releases (standalone) | Marketing content; always corroborate |
| Hospital system "newsrooms" | PR content; verify claims independently |
| Anonymous blog posts | No accountability |
