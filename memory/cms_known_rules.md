## How to Use

Before the Policy Agent runs, check this file. If a rule is listed here, skip re-documenting it **unless**:
- There is a material update (e.g., a proposed rule became final)
- New implementation guidance was issued
- A legal challenge has changed its status

After a run, add any newly documented rules to this file.
